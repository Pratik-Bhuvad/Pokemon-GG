Pokemon Guess Game


Read this Instructions before you Start Game:

  • Pokemon Name - Enter Name of Pokemon given in Image.
      • Each correct answer gives 2 Points.
  • Region - Select region where above Pokemon was First Introduced.
      • Each correct answer gives 1 Points.
  • Type 1 & 2 - Select 2 Types of above Pokemon.
      • If Pokemon is Single type then select second type as NONE.
      Each correct answer(type) give 1 Point.
  • Learnable Moves - Enter any 2 moves which above Pokemon can Learn.
      • Each correct answer(move) give 1 Point.
  • Total 7 Points per Round.
